                        function getWord() {
                          let word = document.getElementById("inputWord").value;
                          word = word.toLowerCase().trim().replace(/[^a-zA-Z0-9 ]/g, "");

                          if (word.length < 2) { 
                            Swal.fire({ 
                              icon: 'error' , 
                              title: 'Oops...' ,
                              text: 'Please enter a word with at least 2 letters.' 
                            }) 
                            return; 
                          } 
                            
                          let revWord=flipWord(word); 
                            
                          displayResults(word, revWord); 
                        } 
                        
                        function flipWord(word) { 
                          let newWord=''; 
                            
                          for (let i=word.length - 1; i>= 0; i--){
                            newWord += word[i];
                          }
                          return newWord;
                        }

                        function displayResults(word, flipped) {
                          let result = document.getElementById('reversedWord');
                          let resultH = document.getElementById('resultH');
                          result.innerHTML = '';

                          result.innerHTML += `<u>${word</u>and the word reversed is<u>${flipped}</u>`;

                          if (word === flipped) {
                            resultH.innerHTML = `<h3>${word}</h3><h4class="text-success">Palindrome!</h4>`;
                            } else {
                            resultH.innerHTML = `<h3>${word}</h3><h4 class="text-danger">Not a Palindrome</h4>`;
                          }
                        }

The code is structured into 3 functions

getWord

getWord gets a string from the id 'inputWord' and saves it to a variable (word) it then utilizes three built in string functions. the first is .toLowerCase(), used to turn all letters lower cased then we use .trim() to remove the white space, and finally we use .replace() to remove unwanted characters. The regular exrpession ignores A-Z upper and lower case and the numbers 0-9. It removes the rest.

The only validation I added was to ensure at least two letters were entered. If that fails, we get an alert explaining why. If it passes, we call flipWord() and pass it our word. it should return a reversed version of our word. Finally, we call displayResults() and pass it our word, and our reversed word for display to the screen.

flipWord

flipWord utilizes a decrementing flor loop to iterate backwards over our word. For each letter it finds, it adds it to a new string. The return value is our new reversed word.
